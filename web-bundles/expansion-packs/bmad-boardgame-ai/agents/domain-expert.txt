# Analyze Game Rule Task

## Purpose
보드게임 룰을 체계적으로 분석하여 명확한 해석을 제공합니다.

## Process
1. **룰 텍스트 확인**
   - 원문 그대로 인용
   - 키워드 하이라이트
   - 관련 용어 정의

2. **문맥 파악**
   - 해당 룰이 속한 게임 단계
   - 연관된 다른 룰과의 관계
   - 게임 전체 흐름에서의 위치

3. **해석 제공**
   - 문자적 해석
   - 의도적 해석
   - 실제 플레이 적용 방법

4. **예시 시나리오**
   - 일반적인 경우
   - 특수한 경우
   - 자주 헷갈리는 상황

5. **참고 자료**
   - 공식 FAQ 확인 사항
   - 커뮤니티 합의 사항
   - 관련 BGG 스레드

## Output Format
```markdown
### 룰 분석: [게임명] - [룰 주제]

**원문:**
> [룰북 원문 인용]

**핵심 키워드:**
- [키워드1]: [정의]
- [키워드2]: [정의]

**해석:**
[상세한 룰 해석]

**적용 예시:**
1. [시나리오 1]
2. [시나리오 2]

**주의사항:**
- [흔한 실수나 오해]

**참고:**
- [관련 FAQ나 커뮤니티 합의]
```

# Evaluate Complexity Task

## Purpose
보드게임 룰 질문의 복잡도를 평가하여 리서치 필요성을 판단합니다.

## Complexity Levels
1. **Simple (간단)** - 점수: 1-3
   - 기본 룰 확인
   - 단순 턴 순서
   - 기초 용어 설명

2. **Medium (중간)** - 점수: 4-6
   - 여러 룰의 상호작용
   - 타이밍 이슈
   - 조건부 효과

3. **Complex (복잡)** - 점수: 7-9
   - 다중 예외 상황
   - 룰 충돌 해결
   - 전략적 판단 필요

4. **Edge Case (엣지케이스)** - 점수: 10+
   - 극히 드문 상황
   - 룰북에 명시되지 않음
   - 공식 FAQ 필요

## Evaluation Criteria
- 관련 룰의 개수
- 예외 상황 포함 여부
- 게임 단계의 복잡성
- 용어의 전문성
- 전략적 판단 필요도

## Output Format
```markdown
### 복잡도 평가 결과

**질문:** [원본 질문]

**복잡도 점수:** [X]/10

**분류:** [Simple/Medium/Complex/Edge Case]

**평가 근거:**
- [근거 1]
- [근거 2]
- [근거 3]

**리서치 권장:** [Yes/No]
- 이유: [리서치가 필요한/불필요한 이유]

**예상 필요 정보:**
- [필요한 정보 유형 1]
- [필요한 정보 유형 2]
```

# Boardgame Glossary

## 아크노바 (Ark Nova) 주요 용어
- **보존 프로젝트**: 특정 조건 달성 시 점수와 보너스를 제공하는 카드
- **후원**: 동물원 운영에 도움을 주는 지원 카드
- **협회**: 플레이어가 가입할 수 있는 조직, 특수 능력 제공
- **동물 태그**: 대륙, 서식지, 크기 등 동물 분류 체계
- **어필**: 방문객 유치력, 수입과 직결
- **보존 점수**: 게임 승리 조건 중 하나
- **휴식**: 사용한 액션 카드를 회수하는 특수 액션

## 글룸헤이븐 (Gloomhaven) 주요 용어
- **선도권**: 라운드 내 행동 순서 결정
- **소진**: 카드를 영구 제거하여 강력한 효과 발생
- **수식어 덱**: 공격 시 데미지 조정 카드
- **개인 목표**: 캐릭터별 은퇴 조건
- **번영도**: 도시 발전 수준, 상점 아이템 해금
- **평판**: 도시에서의 명성, 상점 가격에 영향

## 일반 보드게임 용어
- **액션 포인트**: 턴당 수행 가능한 행동 수
- **엔진 빌딩**: 시너지 효과를 만드는 전략
- **일꾼 놓기**: 말을 놓아 액션을 수행하는 메커니즘
- **드래프팅**: 카드를 순환하며 선택하는 방식
- **자원 관리**: 한정된 자원의 효율적 사용
- **세트 컬렉션**: 특정 조합 모으기

# Complexity Evaluation Criteria

## 점수 배정 기준

### 게임별 특수 용어 (0-3점)
- 일반 용어만 사용: 0점
- 1-2개 특수 용어: 1점
- 3-4개 특수 용어: 2점
- 5개 이상 특수 용어: 3점

### 룰 상호작용 (0-3점)
- 단일 룰: 0점
- 2개 룰 상호작용: 1점
- 3-4개 룰 상호작용: 2점
- 5개 이상 룰 상호작용: 3점

### 타이밍 이슈 (0-2점)
- 타이밍 무관: 0점
- 순서가 중요함: 1점
- 복잡한 타이밍/우선순위: 2점

### 예외 상황 (0-2점)
- 일반적 상황: 0점
- 특수 조건 포함: 1점
- 희귀한 예외 상황: 2점

### 전략적 판단 (0-2점)
- 단순 룰 확인: 0점
- 전략적 고려 필요: 1점
- 복잡한 의사결정: 2점

## 자동 리서치 트리거
다음 키워드 포함 시 자동으로 복잡도 +3점:
- "상충", "충돌", "모순"
- "우선순위", "먼저", "순서"
- "예외", "특수한 경우"
- "FAQ", "에라타", "공식 해석"

## 게임별 복잡도 가중치
- 아크노바: 기본 +1 (복잡한 게임)
- 글룸헤이븐: 기본 +1 (복잡한 게임)
- 윙스팬: 기본 +0 (중간 복잡도)
- 스플렌더: 기본 -1 (간단한 게임)

# Optimize Search Query Task

## Purpose
보드게임 룰 질문에 대한 최적의 검색 쿼리를 생성합니다.

## Query Optimization Process
1. **질문 분석**
   - 핵심 키워드 추출
   - 게임명 정규화
   - 룰 관련 용어 식별

2. **쿼리 구성 전략**
   - 필수 포함 단어 (+)
   - 제외 단어 (-)
   - 구문 검색 ("")
   - 사이트 지정 (site:)

3. **다단계 검색 전략**
   - 1차: 정확한 룰 검색
   - 2차: FAQ/에라타 검색
   - 3차: 커뮤니티 토론 검색

## Query Templates
```markdown
### 기본 쿼리 템플릿
[게임명] +rules +"[특정 룰/카드명]" site:boardgamegeek.com

### FAQ 검색 템플릿
[게임명] FAQ official +"[룰 키워드]" -review

### 커뮤니티 검색 템플릿
[게임명] "how does" "[메커니즘]" site:reddit.com/r/boardgames

### 한국어 검색 템플릿
[게임명] 룰 "[특정 상황]" site:boardlife.co.kr OR site:boardm.co.kr
```

## Optimization Techniques
1. **키워드 변형**
   - 동의어 활용
   - 약어/전체 이름 병행
   - 복수형 고려

2. **부정 키워드**
   - -review -unboxing
   - -"for sale"
   - -video

3. **시간 필터**
   - 최근 1년 내 정보
   - 최신 버전 관련

## Output Format
```markdown
### 최적화된 검색 쿼리

**기본 쿼리:**
```
[최적화된 쿼리 1]
```

**대체 쿼리:**
```
[최적화된 쿼리 2]
```

**확장 쿼리:**
```
[최적화된 쿼리 3]
```

**예상 결과:**
- 정확도: [High/Medium/Low]
- 예상 결과 수: [많음/적당/적음]
- 권장 우선순위: [1/2/3]
```

# Improve Scraping Task

## Purpose
웹 스크래핑 성공률을 높이고 정보 추출 품질을 개선합니다.

## Scraping Optimization
1. **타임아웃 전략**
   - 기본: 3초
   - 복잡한 사이트: 5초
   - 실패 시 재시도: 2회

2. **콘텐츠 선택자 최적화**
   ```javascript
   // BGG 최적화
   const bggSelectors = {
     rules: '.forum-post-body, .wiki-content',
     faq: '.faq-answer, .rules-clarification',
     ignore: '.signature, .ad-content'
   };

   // Reddit 최적화
   const redditSelectors = {
     content: '.comment-body, .post-content',
     score: '.score',
     ignore: '.deleted, .removed'
   };
   ```

3. **노이즈 제거 전략**
   - 광고 블록 필터링
   - 네비게이션 제거
   - 서명/푸터 제거
   - 중복 콘텐츠 제거

## Content Extraction
1. **핵심 문장 추출**
   - 룰 키워드 근처 문장
   - 번호/불릿 포인트 우선
   - 인용문 포함

2. **구조 보존**
   - 리스트 형태 유지
   - 단계별 설명 보존
   - 예시 포함

## Error Handling
```javascript
const scrapeWithFallback = async (url) => {
  try {
    // 1차 시도: 전체 페이지
    return await scrapeFullPage(url);
  } catch (e1) {
    try {
      // 2차 시도: 모바일 버전
      return await scrapeMobileVersion(url);
    } catch (e2) {
      // 3차 시도: 캐시된 버전
      return await scrapeFromCache(url);
    }
  }
};
```

# Boardgame Information Sources

## 주요 영어 소스

### BoardGameGeek (BGG)
- **URL**: boardgamegeek.com
- **특징**: 
  - 가장 권위 있는 보드게임 데이터베이스
  - 공식 FAQ, 에라타 제공
  - 활발한 포럼 토론
- **검색 팁**:
  - `/boardgame/[id]/[game-name]` 구조
  - 포럼은 `/thread/` 경로
  - 룰 문서는 `/filepage/` 경로

### Reddit
- **URL**: reddit.com/r/boardgames
- **특징**:
  - 실시간 커뮤니티 토론
  - 룰 해석 논쟁 활발
  - 초보자 질문 많음
- **하위 레딧**:
  - r/boardgames
  - r/boardgamedesign
  - 게임별 전용 서브레딧

## 주요 한국어 소스

### 보드라이프
- **URL**: boardlife.co.kr
- **특징**:
  - 한국 최대 보드게임 커뮤니티
  - 한글 룰북 공유
  - 국내 플레이어 경험담

### 보드엠
- **URL**: boardm.co.kr
- **특징**:
  - 보드게임 전문 쇼핑몰
  - 공식 한글 룰북
  - 게임 리뷰 및 가이드

### 디스이즈게임
- **URL**: thisisgame.com
- **특징**:
  - 게임 뉴스 및 리뷰
  - 신작 게임 정보
  - 전문가 칼럼

## 신뢰도 평가 기준

### 높음 (공식/준공식)
- 퍼블리셔 공식 사이트
- BGG 공식 FAQ
- 디자이너 직접 답변

### 중간 (커뮤니티 합의)
- BGG 높은 평점 답변
- 여러 사용자 동의
- 유명 리뷰어 해석

### 낮음 (개인 의견)
- 단일 사용자 의견
- 하우스룰 제안
- 추측성 답변

## 검색 우선순위
1. 공식 FAQ/에라타
2. BGG 포럼 (평점 높은 답변)
3. Reddit 토론 (upvote 많은 답변)
4. 한국 커뮤니티 (보드라이프, 보드엠)
5. 개인 블로그/리뷰

# Advanced Search Operators

## Google Search Operators

### 필수 포함/제외
- `+keyword` : 반드시 포함
- `-keyword` : 제외
- `"exact phrase"` : 정확한 구문

### 사이트 지정
- `site:domain.com` : 특정 사이트만
- `site:domain.com/path` : 특정 경로
- `-site:domain.com` : 특정 사이트 제외

### 파일 타입
- `filetype:pdf` : PDF 파일만
- `ext:pdf` : 확장자 지정

### 시간 필터
- `after:2023` : 2023년 이후
- `before:2024` : 2024년 이전

### 와일드카드
- `*` : 임의의 단어
- `game * rule` : game과 rule 사이 임의 단어

## 보드게임 특화 검색 패턴

### 룰 검색
```
"[게임명]" +rules +"[카드/컴포넌트]" -review -unboxing
```

### FAQ 검색
```
"[게임명]" (FAQ OR errata OR clarification) official
```

### 특정 상황 검색
```
"[게임명]" "what happens when" "[specific situation]"
```

### 커뮤니티 의견
```
"[게임명]" "[rule question]" (consensus OR "most people")
```

## 언어별 검색 팁

### 영어 검색
- 복수형 고려 (rule/rules)
- 미국/영국 철자 (color/colour)
- 약어 병행 (FAQ/frequently asked questions)

### 한국어 검색
- 띄어쓰기 변형 고려
- 외래어 표기 차이 (보드게임/보드 게임)
- 줄임말 포함 (아크노바/아크 노바)

## 효율적 검색 전략

### 단계별 접근
1. 넓은 검색 → 좁은 검색
2. 공식 자료 → 커뮤니티 의견
3. 최신 정보 → 과거 토론

### 검색어 조합
- 기본: `게임명 + 핵심 키워드`
- 확장: `게임명 + 동의어 + 사이트 지정`
- 정밀: `"정확한 구문" + 필수 포함 - 제외어`
